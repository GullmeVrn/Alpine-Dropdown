## Getting Started

**1. Clone this repository, go to the root directory and install packages**

```bash
git clone https://github.com/GullmeVrn/Alpine-Dropdown
cd Alpine-Dropdown
npm install
```

**2. Install Parcel globaly.**

```bash
npm install -g parcel
```

**3. Run serve**

```bash
npm run serve
```
